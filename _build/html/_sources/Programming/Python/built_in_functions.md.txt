# Python built-in functions 
Are functions that are always available for use. You don't need to import any module to use them.

[Documentation](https://docs.python.org/3/library/functions.html)  
[Python Standard Library](https://docs.python.org/3/library/index.html)

- `float()` - converts an object into a floating-point number (e.g float(2) will return 2.0)
- `int()` - converts an object into an integer (e.g int(2.0) will return 2)
- `str()` - converts an object into a string (e.g str(2) will return "2")
- `chr()` - converts an integer into a character (e.g chr(65) will return "A")
- `type()` - returns the type of an object (e.g type(2) will return int)
